# Circuit simulation portfolio

**Live portfolio: https://shakc04.github.io/ngspice-portfolio/**

A landing page for two ngspice circuit-simulation projects, served with GitHub Pages.

The first project is a [synchronous buck converter](https://github.com/shakc04/buck-converter-design), a 12 V to 3.3 V, 5 A voltage-mode design with Type III compensation, whose loop gain is measured by Middlebrook injection and whose stability is confirmed across component tolerance by a Monte Carlo study. The second is a [discrete Class-AB amplifier](https://github.com/shakc04/class-ab-amplifier), a complementary Darlington output stage in a feedback loop, characterized for quiescent current, frequency response, distortion against power, slew rate, thermal bias, and output impedance.

The page itself is `index.html` with the headline figures in `assets/`.
